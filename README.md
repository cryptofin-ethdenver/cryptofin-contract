# Diing Contract

## Deployments

### Multichain Deployment with Truffle & Infura
```angular2html
# .env File
DEPLOYER_WALLET_PRIVATEKEY=######
DEPLOYER_WALLET_ADDRESS=#######
INFURA_API_KEY=########
```

```
$ npm install -g truffle
$ truffle deploy --network {network}
```
